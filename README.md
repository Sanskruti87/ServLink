**Project Description**

ServLink Pro is a full-stack web application that connects customers with local service providers
such as electricians, plumbers, cleaners, and carpenters. Customers can easily book services through 
the platform, while service providers can view, accept, or reject booking requests from their dashboard.
The project is built as a Minimum Viable Product (MVP) demonstrating a complete booking workflow using the MERN stack.

**Features**

1.Authentication
- User registration and login
- JWT authentication
- Password hashing using bcrypt
- Role-based access (Customer / Provider)

2.Customer Dashboard
- View available services
- Book a service
- View booking history
- Simulated payment options

3.Provider Dashboard
- View incoming booking requests
- Accept or reject jobs
- Track accepted jobs

4.Booking System
- Customers create service bookings
- Providers receive booking requests
- Booking status updates in the database

5.Payment Simulation
Mock payment methods:
- UPI
- Card
- Cash
